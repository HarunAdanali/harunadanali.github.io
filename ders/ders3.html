Listeler

HTML'da üç tür liste vardır.
  1. Sıralı Listeler
  2. Sırasız Listeler
  3. Tanımlama Listeleri

Sıralı Listeler <ol></ol> (Ordered List)

<ol>
    <li>İlk bahar</li>
    <li>Kış</li>
    <li>Sonbahar</li>
    <li>Yaz</li>
</ol>
Bak Böyle

<ol> etiketinde type ve start parametleri kullanılabilir 
<ol type="I" start="2">
    <li>madde</li>
    <li>zararsız madde</li>
    <li>bağımlılık yapmayan</li>
    <li>su gibi</li>
</ol>
Type için a,A,i,I,1 değerleri kullanılabilir
Start için kaçıncı maddeden başlamak istiyorsanız o değerden başlayabilirsiniz

Ayrıca type maddeler içinde de tanımlanabilir. 
<ol>
    <li type="A">madde 1</li>
    <li type="a">madde 2</li>
    <li type="I">madde 3</li>
    <li type="i">madde 4</li>
    <li type="1">madde 5</li>
</ol>

Sırasız Listeler <ul></ul> (Unordered List) 

<ul>
    <li>Çay</li>
    <li>Kahve</li>
    <li>Meyve Suyu</li>
</ul>
bu etiketde de type (disc,circle,square) değerlerini alınabilir. 
<ul>
    <li type="disc">elma</li>
    <li type="circle">armut</li>
    <li type="square">kel mahmut</li>
</ul>

Sayfa içi Tanım

head bloğunda <style></style> etiketleri arasında tanımlanır. 

<html>

<head>
    <title> CSS işte Çok DA Abarmamak Lazım </title>
    <style>
        p {color: blue;font-family: verdana;font-size: 10pt;}

        h1 {color: rgb(25, 210, 0);}
    </style>

</head>

<body>
    <h1>Bu başlık yeşil ne kadar gıcık bi rek ya</h1>
    <p>bu yazı mavi verdana ne 10 punto</p>
</body>

</html>

Satır İçi Tanım

Satı içerisinde style parametresiyle tanımlanır. 

<html>
    <head>
        <title>  </title>
        <style>
            p{color: red;}
        </style>
    </head>
    <body>
        <h1 style="color: green; border: 5px solid blue;">Burası başlık</h1>
        <p style="color: orange;font-weight: bold;">burası verdana,turuncu ve bold</p>
    </body>
</html>

CSS Tanımlamada temel selektör yapıları

Selektör Örnek Etki Alanı
Element a{özellik} Tüm a etiketleri
Element,Element p,div{özellik} Tüm paragraf ve div etiketleri
.class .kutu{özellik} class="kutu" olarak tanımlanan heryer
#id #renk{özellik} id="renk"şeklinde tanımlanan her yer
* *{özellik} Tüm etiketler
